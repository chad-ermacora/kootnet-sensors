Sensor Community (Luftdaten)







Information

You need to add your Raspberry Pi's "Sensor ID" shown on this page to your Sensor Community account. If you are unable to add the Sensor ID to your Sensor Community account, try enabling Sensor Community here first.

You must have at least one of the following sensors
Pimoroni EnviroPhat / Enviro+, BMP280, BME680, PMS5003, Sensirion SPS30

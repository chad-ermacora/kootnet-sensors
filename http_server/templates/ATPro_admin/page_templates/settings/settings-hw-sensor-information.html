Supported Hardware Information

If you do not have any sensors yet, you can always enable the "Kootnet Dummy Sensors", this will provide randomly generated fake readings for all supported sensor types. Even if you have some sensors, you can still enable the "Kootnet Dummy Sensors", your real sensors will override the dummy sensors.

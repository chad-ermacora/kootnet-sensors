Introduction #back to top

Kootnet Sensors is a program used to take sensor readings from multiple sensors and record them to an SQLite3 database and or upload them to one or more Online 3rd party service like Open Sense Map. Add one or more of the 25+ supported hardware sensors to record and view readings of your choice. Sensor types include heat, pressure, altitude, humidity, distance, gas, particulate matter 1/2.5/4/10, light, color, ultra violet, acceleration, electromagnetic fields and angular velocity (gyroscope).

KootNet Sensors requires a Raspberry Pi to utilize Sensors but can be installed on Ubuntu or Debian in order to use the “Sensor Control” web interface for managing up to 20 other remote Kootnet Sensors as well as uploading other sensor databases in order to utilize more powerful computers in creating graphs.

Installation & Removal #back to top

Requirements

Operating System : Ubuntu, Debian, Raspbian/Raspberry Pi OS
Other Requirements : Internet Access
Note : Most installs finish within a few minutes but the Raspberry Pi Zero can take up to 20 minutes

Installation using a GUI Desktop

Installation from a supported GUI based Operating system is fairly straight forward. Click one of the following links to download Kootnet sensors, then double click the file in your operating systems file manager to install.

Current Stable Release
Latest Developmental Release ** Warning: Developmental Releases can be unstable **

Once the install has completed, you can find shortcuts to the various program function in your operating systems menu.

Installation using a Terminal

To install using a terminal, enter the following command, hit enter and wait for it to finish.

wget -O KootnetSensors.deb http://kootenay-networks.com/installers/KootnetSensors.deb && sudo apt-get update && sudo apt-get -y install ./KootnetSensors.deb

If the installation completes successfully, you'll be given the URL to access the sensor.

To remove Kootnet Sensors, you can find it in your Operating Systems Application Manager or run the following Terminal command.

sudo apt-get remove kootnet-sensors
